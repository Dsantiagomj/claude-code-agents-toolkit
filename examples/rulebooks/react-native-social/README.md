# React Native Social Media Example

Real-world social media app with 500,000+ active users.

## Project Context

- **Users:** 500,000+ active
- **Scale:** 2M+ posts/month
- **Platforms:** iOS & Android (single codebase)
- **Team:** 8 mobile developers

## What's Unique

- **Offline-First:** WatermelonDB for local storage & sync
- **Real-Time Chat:** Firebase Realtime Database
- **Push Notifications:** Expo Notifications + FCM
- **Camera Integration:** In-app camera with filters
- **Deep Linking:** Universal links for sharing
- **Performance:** FlashList for smooth scrolling

## Key Decisions

1. **Expo vs Bare RN:** Expo (faster dev, OTA updates)
2. **WatermelonDB vs AsyncStorage:** WatermelonDB (10K+ posts)
3. **Firebase vs Custom:** Firebase (real-time out-of-box)

## Use This Example When

✅ Building social media apps
✅ Need offline-first architecture
✅ Require real-time chat
✅ Want push notification patterns
✅ Need camera/media handling
