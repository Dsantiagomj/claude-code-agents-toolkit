# React Admin Dashboard Template

> Pre-configured RULEBOOK for admin panels and analytics dashboards built with React, Redux Toolkit, and Material-UI.

## What's Included

### Tech Stack
- **React 19** (Vite) - Modern React with latest features
- **TypeScript 5.5+** - Type-safe development
- **Material-UI v6** - Professional UI components
- **Redux Toolkit** - Powerful state management
- **Chart.js** - Beautiful data visualization
- **TanStack Table** - Advanced data tables
- **RTK Query** - Efficient API integration
- **Vitest + Playwright** - Complete testing suite

### Key Features
✅ Real-time data updates (WebSocket)
✅ Complex data visualization (charts, graphs)
✅ Advanced filtering & search
✅ Role-based access control (RBAC)
✅ Export functionality (PDF, CSV, Excel)
✅ Dark mode support
✅ Responsive design
✅ Feature-based architecture
