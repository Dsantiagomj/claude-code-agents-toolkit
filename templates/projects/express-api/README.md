# Express REST API Template

> Pre-configured RULEBOOK for production-ready REST APIs built with Express.js, MongoDB, and TypeScript.

## What's Included

### Tech Stack
- **Express.js 5.x** - Fast, unopinionated web framework
- **TypeScript 5.5+** - Type-safe backend development
- **MongoDB 7.x** - NoSQL database
- **Mongoose 8.x** - Elegant MongoDB ODM
- **JWT** - Secure authentication
- **Swagger/OpenAPI** - Automatic API documentation
- **Jest + Supertest** - Comprehensive testing

### Key Features
✅ RESTful API design principles
✅ JWT authentication & authorization
✅ Input validation (Zod)
✅ Rate limiting & security (Helmet, CORS)
✅ Comprehensive error handling
✅ API documentation (Swagger UI)
✅ File upload support (S3)
✅ Logging (Winston) & monitoring (Sentry)

## When to Use This Template

✅ **Perfect For:**
- REST API services
- Microservices backends
- Mobile app backends
- SaaS API layers
- Internal APIs

❌ **Not Ideal For:**
- GraphQL APIs (use NestJS instead)
- Real-time apps only (add Socket.io)
- Serverless functions (use AWS Lambda)
