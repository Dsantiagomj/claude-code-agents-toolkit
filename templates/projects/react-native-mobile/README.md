# React Native Mobile App Template

> Pre-configured RULEBOOK for cross-platform mobile applications built with React Native and Expo.

## What's Included

### Tech Stack
- **React Native 0.74+** - Cross-platform mobile framework
- **Expo SDK 52** - Managed workflow & native APIs
- **TypeScript 5.5+** - Type-safe mobile development
- **React Navigation 6.x** - Native navigation
- **Zustand + React Query** - State & data management
- **WatermelonDB** - Offline-first database
- **Firebase** - Authentication & push notifications
- **Jest + Maestro** - Comprehensive testing

### Key Features
✅ Cross-platform (iOS + Android)
✅ Native performance
✅ Offline-first architecture
✅ Push notifications
✅ Deep linking
✅ Biometric authentication
✅ Image optimization
✅ Analytics & crash reporting
